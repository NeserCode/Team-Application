# Team Application

## 应用概述

Team 是一个基于 Vue.js 与 Electron 的 Windows 桌面应用。

提供了 Leetcode 刷题的解决方案，同时共享同一区域内所有人的有效提交记录。以加强互相之间解题思路的交流。

## 为什么制作 Team

🏓 磨练自己的技术

🎃 尝试新的领域与框架（Electron）

⚽ 为自己实验室下的伙伴提供更好的考勤与日常刷题的解决方案
